## Variáveis de ambiente

Essas variáveis de ambiente são necessárias para rodar o projeto:

```env
API_URL
PORT
DEBUG
USER
TOKEN
```

- `API_URL`: URL da API que estamos integrando
- `PORT`: Porta HTTP que a app vai subir (padrão `8080`)
- `DEBUG`: indica se a app deve ser iniciada em modo de debug (padrão `false`)
- `USER`: usuário da API que estamos integrando
- `TOKEN`: token de integração do usuáiro que estamos integrando
